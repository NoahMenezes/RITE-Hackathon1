# FOCUSFLOW.PROTOCOL

The systematic deconstruction of modern workspace architecture. Built for high-performance practitioners.

## 🏁 CORE STACK
- **Engine**: Next.js 16.2.1 (Turbopack)
- **Neural Layer**: Supabase Authentication (SSR-enabled)
- **Visual Grid**: Tailwind CSS v4
- **Dynamics**: Framer Motion
- **UI Architecture**: custom Magic UI / ShineBorder / AnimatedList
- **Package Control**: pnpm

## ⚡ ARCHITECTURE OVERVIEW

### `/dashboard` (The Command Center)
Secure identity verification required. A minimalist environment designed for maximum cognitive output. Protected by edge-level proxy protocols.

### `/team` (The Mesh Network)
Transparent communication channel for the core engineering and design team. Features an encrypted transmission form for direct hub inquiries.

### `/` (The Manifesto)
Manifesto detailing the evolution of flow. Systematics of workplace deconstruction through high-end industrial optics.

## 🔒 SECURITY & AUTHENTICATION
Identity verification is strictly enforced via Supabase SSR.
- **Edge Security**: Native proxy routing ensures no unauthenticated packets reach protected modules.
- **Encryption**: Standard-grade password encryption and secret key verification protocols.

## 🎨 DESIGN SYSTEM
Strict industrial minimalist aesthetic:
- **Zero Radius**: Total elimination of rounded geometric edges (`0 !important`).
- **High Contrast**: Monochromatic foundational palette with elective electric blue accents.
- **Motion Logic**: Spring-based physics for all UI transitions.
